🎬 Netflix-Core: High-Throughput Media Ingestion & HLS Pipeline
Developed by: Prem Bashyal

An enterprise-grade media processing and adaptive bitrate streaming microservice architecture simulation built using Python and FFmpeg core.

🛠️ Key Architectural Strengths
Vectorized Transcoding: Implements multi-threaded FFmpeg processing rules to split raw MP4 files into HLS (HTTP Live Streaming) chunks (.ts segments and .m3u8 master playlists).
Adaptive Bitrate Ready: Designed to simulate smooth, buffering-free client playback across varying network bandwidth parameters.
Microservices Interface: Served via an optimized modern dark-themed dashboard using Streamlit.
